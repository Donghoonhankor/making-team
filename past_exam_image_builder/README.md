# 기출이미지생성기

기존 수학도표렌더러와 분리된 기출 그림 등록 및 수치 교체 도구입니다.

## PDF 일괄 등록

1. `PDF 일괄 분석·등록` 탭에서 `PDF 선택 및 자동 분석`
2. 파일명에서 학교, 연도, 학년, 학기, 시험구분을 자동 입력
3. 그림 후보와 추천 문제번호 확인
4. 잘못된 문제번호는 표에서 직접 수정
5. 불필요한 후보는 체크 해제
6. `체크한 그림 일괄 등록`

답지와 해설지 페이지는 자동으로 제외합니다. 스캔 PDF에서 큰 그림 객체를 찾아
문항별 라이브러리에 등록합니다.

## 개별 등록

1. `PDF 열기`
2. 문제 그림이 있는 페이지로 이동
3. 그림 전체를 마우스로 드래그
4. `선택 영역 미리보기`
5. 바꿀 숫자가 있으면 해당 숫자 영역을 드래그하고 `선택 영역을 숫자 변수로 추가`
6. 학교, 연도, 학년, 학기, 시험구분, 문제번호 확인
7. `문제 이미지 등록`

등록 자료는 실행 파일 옆 `library` 폴더에 저장됩니다. 자동 분석에서 놓친 그림이나
영역을 정밀하게 수정해야 할 때 개별 등록을 사용합니다.

## 생성

1. `수치 변경 생성` 탭
2. 문제 ID 선택
3. 새 숫자 입력
4. `새 PNG 생성`

원본 그림에서 등록한 숫자 영역만 흰색으로 지운 뒤 새 값을 같은 위치에 출력합니다.
